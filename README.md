## 📌 Project Vision

This project explores the **Titanic passenger dataset** through advanced **Exploratory Data Analysis (EDA)** techniques to uncover meaningful survival patterns, passenger trends, and socio-economic insights.

By transforming raw historical data into actionable intelligence, this project demonstrates practical data analytics skills used in modern business and decision-making environments.

## 📊 Strategic Insights Discovered

### 🔹 Survival by Gender
- Female passengers had a significantly higher survival probability (**~74%**)  
- Male passengers showed a much lower survival rate (**~19%**)

### 🔹 Passenger Class Impact
- Travelers in **1st Class** had noticeably stronger survival outcomes than lower classes.

### 🔹 Economic Influence
- Higher ticket fares were positively associated with improved survival chances.

### 🔹 Hidden Patterns
- Socio-economic status and demographic factors played a major role in rescue outcomes.
